## Example Project

This project demonstrates how to use the `face_aisdk_flutter_plugin`.

### How to Run

1.  **Get dependencies**:
    ```bash
    flutter pub get
    ```
2.  **Run the app**:
    ```bash
    flutter run
    ```

### Notes
- Make sure you have a physical device connected, as camera features might not work on simulators.
- For Android, ensure you have the necessary SDK licenses if required by the native FaceAISDK.

## 示例项目 (简体中文)

本项目展示了如何使用 `face_aisdk_flutter_plugin` 插件。

### 如何运行

1.  **获取依赖**:
    ```bash
    flutter pub get
    ```
2.  **运行项目**:
    ```bash
    flutter run
    ```

### 注意事项
- 请确保连接了真机设备，因为模拟器可能无法正常使用相机功能。
- 对于 Android 端，请确保已按照原生 FaceAISDK 的要求配置了必要的 SDK 授权/许可。
